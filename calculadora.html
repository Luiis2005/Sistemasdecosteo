Calculadora de Depreciación

Método de Línea Recta (LR)

Datos del Activo

Dt = (B - S) × dt
Dt = (B - S) / n
Donde:
• Dt = Cargo anual de depreciación
• B = Costo inicial o base no ajustada
• S = Valor de rescate estimado
• n = Periodo de recuperación (años)
• dt = Tasa de depreciación = 1/n
